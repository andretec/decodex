    <article class="markdown-body entry-content" itemprop="mainContentOfPage"><h1><a id="user-content-convertstl" class="anchor" href="#convertstl" aria-hidden="true"><span class="octicon octicon-link"></span></a>convertSTL</h1>

<p>A utility for converting STL files between ASCII and binary encoding.
I wrote this to save disk space and bandwidth when handling STL files. Some modeling programs include this functionality, but it is nice to have something that is more lightweight and can be called from a command line.</p>

<h2><a id="user-content-usage-" class="anchor" href="#usage-" aria-hidden="true"><span class="octicon octicon-link"></span></a>USAGE </h2>

<pre><code>$ ruby convertSTL.rb [filename of .stl to be converted]
</code></pre>

<p>or 'chmod +x' the script and run as:</p>

<pre><code>$ ./convertSTL.rb [filename of .stl to be converted]
</code></pre>

<p>The script will then translate the STL to the opposite encoding and save it as either <code>-ascii.stl</code> or <code>-binary.stl</code></p>
